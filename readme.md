# Zingo Game

![Zingo Logo](PNG/logo.png)

## Description

Zingo is a fun and interactive game developed using Python and Pygame. The game involves players drawing cards and placing them on their respective tables. The first player to fill their table wins the game.

## Features

- **Multiple Players**: Supports up to 5 players.
- **Interactive Graphics**: Uses Pygame for rendering graphics and handling user input.
- **Sound Effects**: Includes various sound effects for different game actions.
